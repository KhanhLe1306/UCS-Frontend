PKI Classroom Scheduler

About This Application

The PKI Classroom Scheduler is set to be a state of the art, bleeding edge, epic web application used
by the University of Nebraska at Omaha for all of it's classroom scheduling needs (within PKI that is . . . for now).

Here's how you use it

  • To Search
    for what classes are in a room, or what classes an instructor is currently teaching, simply type the room number or instructor's full name into the search bar.
  • To Add
    a new class: Select the add class button on the home page, enter the required fields, then press the submit button.
  • To Add
    a new instructor: Select the add instructor button on the home page, enter the required fields, then press the submit button.
  • To Remove
    a course from the schedule: Search for the class you want to remove (either by room number or instructor name), click on the class you want to remove, select the remove button.
  • To Upload
    the current schedule as CSV: Go to the other options page, select the "Upload Current Schedule as CSV" button, select your file, select the upload button.
  • To Download
    the updated schedule as CSV: Go to the other options page, select the "Download Current Schedule as CSV" button.
  • To Logout
    Go to the other options page, select the log out button.
